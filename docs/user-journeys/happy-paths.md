# SmartWheels User Journey Happy Paths

## Overview
SmartWheels (ThemeFaaast) is a premium website theme and digital solution service offering custom themes, development, performance optimization, code refactoring, bug fixes, and template marketplace.

## User Types and Happy Paths

### 1. Non-Signed In User

#### Primary Goal: Get a Quote for Digital Services
**Happy Path: First-time visitor discovers service and requests quote**

1. **Discovery Phase**
   - User lands on homepage via search/ads/social media
   - Views hero section with "Your Brand Specialist" messaging
   - Sees statistics: 5000+ websites serviced, 15+ years experience, 100% satisfaction
   - Browses service cards: Theme Design, Custom Development, Performance Optimization, etc.
   - Views portfolio gallery showcasing before/after transformations

2. **Service Exploration**
   - Clicks "View Our Services" button
   - Reviews detailed service descriptions and features
   - Learns about service capabilities (RAL colors, precision machining, etc.)
   - May explore "View Inventory" link to marketplace

3. **Quote Request Journey**
   - Clicks "Request a Quote" or "Start Your Quote Today" CTA
   - Fills out comprehensive quote form:
     - Contact info (name, email, phone)
     - Service selection (checkboxes for multiple services)
     - Project specs (scope, timeline, budget)
     - Photo uploads (up to 5 images of current website/design)
     - Detailed description of requirements
   - Submits form successfully
   - Receives confirmation and next steps via email

#### Alternative Happy Path: Browse and Purchase Themes
**Happy Path: User wants to buy themes**

1. **Marketplace Discovery**
   - Clicks "View Inventory" on homepage or services page
   - Browses external marketplace (marktplaats.nl)
   - Views available themes by category, style, features

2. **Purchase Process**
   - Contacts seller via marketplace platform
   - Negotiates price and arranges pickup/delivery
   - Completes transaction through marketplace

#### Alternative Happy Path: Get Contact Information
**Happy Path: User needs immediate contact**

1. **Contact Page Visit**
   - Clicks "Contact" in navigation
   - Views contact details: address, email, phone, hours
   - Sees map placeholder and directions
   - Calls or emails directly for appointment

### 2. Signed-In (Registered) User

#### Primary Goal: Manage Quotes and Track Services
**Happy Path: Returning customer with account**

1. **Authentication**
   - Signs in via login page with email/password
   - Redirected to profile/dashboard (assumed functionality)

2. **Service Management**
   - Views previous quotes and service history
   - Tracks current service status/progress
   - Manages contact preferences

3. **New Service Request**
   - Accesses streamlined quote form (pre-filled contact info)
   - Uploads photos and specifies requirements
   - Receives faster quote turnaround due to account history

#### Alternative Happy Path: Repeat Business
**Happy Path: Satisfied customer returns for additional services**

1. **Account Login**
   - Signs in to existing account
   - Views service history and past quotes

2. **Quick Quote Request**
   - Uses account data to speed up quote process
   - References previous work for consistent quality
   - May get priority scheduling

### 3. Returning Customer

#### Primary Goal: Trust-Based Service Request
**Happy Path: Previous customer returns based on satisfaction**

1. **Direct Access**
   - Visits website with brand recognition
   - Remembers positive experience from previous service

2. **Streamlined Process**
   - May use direct contact methods (phone/email)
   - References previous work when requesting quote
   - Gets recognized service based on history

3. **Loyalty Benefits**
   - May receive preferential treatment
   - Faster quote turnaround
   - Priority scheduling for busy periods

### 4. Website Builder User (Business Owner)

#### Primary Goal: Customize Website with Theme and Content
**Happy Path: Business owner creates custom website pages**

1. **Theme Studio Access**
   - Signs in and navigates to Theme Studio
   - Explores color, spacing, and typography customization
   - Saves custom theme with brand colors

2. **Component Selection**
   - Switches to "Website" tab in Theme Studio
   - Browses 25+ available components by category
   - Selects relevant components (hero, services, testimonials, contact form)
   - Uses search and filtering to find specific components

3. **Page Creation Process**
   - Creates up to 3 custom pages with descriptive titles
   - Adds up to 5 components per page with drag-and-drop
   - Automatically generates SEO-friendly URLs (/pages/services)
   - Previews page layout and component arrangement

4. **Navigation Integration**
   - Newly created pages automatically appear in site navigation
   - Updates both desktop and mobile navigation menus
   - Pages accessible immediately after creation
   - Clean, professional URLs for each custom page

5. **Live Website Experience**
   - Custom pages render with selected theme
   - Components display placeholder content appropriately
   - Responsive design works across all devices
   - Navigation provides seamless user experience

#### Alternative Happy Path: Content Management
**Happy Path: Updating existing pages with new components**

1. **Page Management**
   - Accesses existing pages in website builder
   - Adds new components to enhance page functionality
   - Reorders components for better user experience
   - Removes outdated components

2. **Content Optimization**
   - Selects components that improve conversion (CTA buttons, testimonials)
   - Organizes content flow for better user engagement
   - Maintains consistent branding across all pages

---

### 5. Purchase Customer (Theme Buyer)

#### Primary Goal: Find and Purchase Quality Themes
**Happy Path: Customer looking to buy themes**

1. **Marketplace Access**
   - Discovers marketplace link on homepage/services
   - Browses inventory on external platform (marktplaats.nl)

2. **Selection Process**
   - Filters by size, brand, condition, price
   - Views detailed photos and descriptions
   - Compares multiple options

3. **Purchase Completion**
   - Contacts seller through marketplace
   - Negotiates and completes transaction
   - Arranges pickup/delivery logistics

#### Alternative Happy Path: Sell Themes
**Happy Path: Customer wants to sell custom themes**

1. **Marketplace Engagement**
   - Accesses marketplace for selling
   - Lists themes with demos and specifications
   - Sets competitive pricing based on market knowledge

2. **Sale Management**
   - Responds to buyer inquiries
   - Negotiates fair prices
   - Completes transactions through platform
